# V2V-C-V2X-vehicular-networks-
Production-grade detail, and implemented in simulation a formally verified cooperative right-of-way negotiation protocol for C-V2X vehicular networks — the missing protocol layer above every V2X standard deployed today
Handshake is a production-grade, safety-certified cooperative negotiation protocol for vehicle-to-vehicle (V2V) and vehicle-to-everything (V2X) communication. It enables every type of vehicle — from a manually driven car to a fully autonomous freight truck — to exchange intent, negotiate right-of-way, protect vulnerable road users, and coordinate movement in real time, with or without any infrastructure support.
This document is the complete engineering blueprint. It supersedes all prior documents and incorporates every technical gap previously identified. It is structured so that a competent engineering team can build Handshake directly from this specification, and an ASIL-D certification auditor can use it as the primary safety evidence document.
